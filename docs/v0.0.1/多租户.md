# 多租户需求文档

## 一、模块概述

多租户是 SaaS 平台的基础能力，支持多个租户在同一系统中独立运营，数据完全隔离。

## 二、核心概念

| 概念 | 说明 |
|------|------|
| 租户 Tenant | 独立的运营主体，拥有独立的空间 |
| 门户 Portal | 租户的前台展示站点 |
| 管理后台 Admin | 租户的后台管理系统 |
| 小程序 MiniApp | 租户的小程序应用 |

## 四、数据隔离方案

### 4.1 逻辑隔离

- 每个租户有唯一的 `tenant_id`
- 所有数据表包含 `tenant_id` 字段
- 查询时自动注入租户 ID

### 4.2 缓存隔离

- Redis key 添加租户前缀
- 租户间缓存完全隔离
