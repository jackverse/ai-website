# AI 生成页面设计

## 一、页面布局

默认**左右两栏布局**，代码编辑器作为小众功能**默认隐藏**，点击展开。

```
┌─────────────────────────────────────────────────────────────────────────────────────────────┐
│  AI 生成                                                    [显示代码编辑器 ▼]              │
├────────────────┬────────────────────────────────────────────────────────────────────────┤
│                │                                                                        │
│  ① 需求输入     │                    ② 预览区域                                        │
│                │                    (实时渲染)                                           │
│  · 需求描述     │                                                                        │
│  · 页面类型     │                    ┌───────────────────────────────────────────────┐   │
│  · 风格选择     │                    │                                               │   │
│  · 参考图上传    │                    │                                               │   │
│                │                    │              渲染效果展示                        │   │
│  ┌──────────┐  │                    │                                               │   │
│  │ 发送生成  │  │                    │                                               │   │
│  └──────────┘  │                    │                                               │   │
│                │                    │                                               │   │
│  ┌──────────┐  │                    └───────────────────────────────────────────────┘   │
│  │ 历史记录  │  │                                                                        │
│  └──────────┘  │                    设备切换: [PC] [平板] [手机]                           │
│                │                    [刷新预览] [全屏] [导出]                              │
├────────────────┴────────────────────────────────────────────────────────────────────────┤
│  ③ 代码编辑器 (默认折叠，点击"显示代码编辑器"展开)                                        │
│  ┌─────────────────────────────────────────────────────────────────────────────────────┐ │
│  │ <template>... │ <script setup>... │ <style>...                                      │ │
│  │                                                                                     │ │
│  │  [格式化] [复制] [保存模板]                                                           │ │
│  └─────────────────────────────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────────────────────────────┘
```

**交互说明**：
- 默认只有左右两栏（需求输入 + 预览）
- 代码编辑器默认隐藏，点击「显示代码编辑器」展开
- 展开后可关闭重新隐藏
- 满足绝大多数用户"输入需求 → 预览"的使用场景

## 三、工作模式

### 模式一：AI 生成模式（默认）

```
用户输入需求 → 发送 → AI 生成代码 → 自动预览
```

### 模式二：手动编辑模式（需展开编辑器）

```
点击"显示代码编辑器" → 展开编辑器 → 直接编辑代码 → 实时预览
```

## 四、交互细节

### 4.1 发送生成

1. 用户在左侧输入需求描述
2. 选择页面类型、风格等选项
3. 点击「发送生成」按钮
4. 显示加载状态（生成中...）
5. AI 返回代码，预览自动刷新

### 4.2 代码编辑（展开后）

1. 点击「显示代码编辑器」展开面板
2. 用户可在 Monaco Editor 中编辑代码
3. 编辑后自动触发防抖预览更新（300ms）
4. 支持快捷键：
   - `Ctrl+S`: 保存模板
   - `Ctrl+Shift+F`: 格式化代码
   - `Ctrl+Z`: 撤销
   - `Ctrl+Y`: 重做

### 4.3 预览更新

1. **自动更新**: 代码变化后 300ms 防抖自动刷新预览
2. **手动刷新**: 点击刷新按钮强制刷新
3. **设备切换**: 切换设备后自动调整预览尺寸

## 五、技术实现

### 5.1 前端技术

| 技术 | 用途 |
|------|------|
| Vue 3 | 页面框架 |
| TypeScript | 类型安全 |
| Tailwind CSS | 样式方案 |
| Headless UI | 组件库 |
| Monaco Editor | 代码编辑器（可选） |
| Vite | 构建工具 |

### 5.2 预览渲染方案

| 方案 | 说明 |
|------|------|
| iframe 渲染 | 安全隔离，推荐 |
| Shadow DOM | 样式隔离 |
| 动态组件 | 直接渲染 Vue 组件 |

**推荐方案**: iframe 渲染
- 代码存储到内存/数据库
- 通过 blob URL 在 iframe 中渲染
- 避免样式冲突

### 5.3 API 接口

| 接口 | 说明 |
|------|------|
| POST /ai/generate | 发送需求，生成代码 |
| GET /ai/generate/{id} | 获取生成结果 |
| POST /ai/template/save | 保存为模板 |
| GET /ai/template/list | 获取模板列表 |

## 六、组件结构

```vue
<template>
  <div class="ai-generator">
    <!-- 左侧：需求输入 -->
    <div class="left-panel">
      <PromptInput />
      <HistoryList />
    </div>

    <!-- 右侧：预览 -->
    <div class="right-panel">
      <PreviewFrame :code="code" />
      <DeviceSwitcher v-model="device" />
      <PreviewToolbar @refresh="refresh" @fullscreen="fullscreen" />
    </div>

    <!-- 中间：代码编辑器（默认隐藏，点击展开） -->
    <div class="center-panel" v-if="showEditor">
      <MonacoEditor v-model="code" />
      <EditorToolbar @format="format" @copy="copy" @save="saveTemplate" />
    </div>

    <!-- 展开编辑器按钮 -->
    <button @click="showEditor = !showEditor">
      {{ showEditor ? '隐藏代码编辑器' : '显示代码编辑器' }}
    </button>
  </div>
</template>
```

## 七、状态管理

```typescript
interface GeneratorState {
  // 需求
  prompt: string;
  pageType: 'home' | 'list' | 'detail' | 'form';
  style: 'tech' | 'simple' | 'business';
  referenceImage?: File;

  // 代码
  code: string;
  isGenerating: boolean;
  showEditor: boolean;  // 代码编辑器是否显示，默认 false

  // 预览
  device: 'pc' | 'tablet' | 'mobile';
  previewKey: number; // 用于强制刷新预览

  // 历史
  historyList: PromptHistory[];
}
```
