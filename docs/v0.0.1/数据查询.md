# 数据查询需求文档

## 一、模块概述

数据查询为租户提供简单的数据库查询能力，用于数据预览和分析。

## 二、功能清单

### 2.1 数据浏览

| 功能点 | 描述 | 优先级 |
|--------|------|--------|
| 数据表列表 | 查看可查询的数据表 | P0 |
| 表结构查看 | 查看表字段结构 | P0 |
| 数据预览 | 查看表数据 | P0 |
| 分页浏览 | 分页查看数据 | P0 |

### 2.2 简单查询

| 功能点 | 描述 | 优先级 |
|--------|------|--------|
| 条件查询 | 支持简单的WHERE条件 | P0 |
| 字段选择 | 选择要查询的字段 | P1 |
| 排序配置 | 配置排序规则 | P1 |
| 结果限制 | 限制返回条数 | P0 |

### 2.3 数据导出

| 功能点 | 描述 | 优先级 |
|--------|------|--------|
| Excel 导出 | 导出为 Excel 文件 | P1 |
| CSV 导出 | 导出为 CSV 文件 | P1 |
| JSON 导出 | 导出为 JSON 文件 | P1 |

### 2.4 查询权限

| 功能点 | 描述 | 优先级 |
|--------|------|--------|
| 表级权限 | 控制可查询的表 | P0 |
| 字段权限 | 控制可查询的字段 | P1 |
| 操作权限 | 控制导出等操作权限 | P0 |

## 三、安全限制

> **核心原则：只读不能修改**

- 仅支持 SELECT 查询
- 不支持 UPDATE/DELETE/DROP 等写操作
- 不支持跨库查询
- 查询超时限制 30 秒
- 单次查询结果限制 1000 条

## 四、界面设计

### 4.1 查询界面

```
+----------------------------------+
| 表选择  | 表A ▼                    |
+----------------------------------+
| 字段    | ☑ 字段1 ☑ 字段2         |

| 条件    | 字段1 = 'xxx'           |
+----------------------------------+
| 排序    | 字段1 ↓                  |
+----------------------------------+
|         [查询] [导出]             |
+----------------------------------+
| 结果预览表格                     |
+----------------------------------+
```

## 五、审计日志

- 记录所有查询操作
- 记录查询用户、查询时间、查询语句
- 支持日志导出
