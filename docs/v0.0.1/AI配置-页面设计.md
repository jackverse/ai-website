# AI 配置 - 页面设计（ai-editor-web）

## 一、设计原则

1. **一致性**：与 renren-ui 风格保持一致
2. **组件化**：使用 Headless UI 组件库
3. **响应式**：支持桌面端操作，移动端可查看
4. **用户友好**：清晰的交互流程，即时反馈

## 二、页面结构

| 页面模块 | 路由 | 说明 |
|----------|------|------|
| 首页 | `/dashboard` | 欢迎页、快捷入口 |
| 项目管理 | `/project` | 项目列表、创建、编辑 |
| 项目版本 | `/project/:id/version` | 版本管理 |
| 模板库 | `/template` | 模板列表、详情、使用 |
| 页面管理 | `/page` | 页面列表、编辑、发布 |
| AI 生成 | `/generate` | 输入需求、生成页面 |
| 组件管理 | `/component` | 组件库、组件配置 |
| 分类标签 | `/category` | 分类管理、标签管理 |
| 成本控制 | `/quota` | 配额查看、使用统计 |
| 提示词优化 | `/prompt` | 日志查看、统计分析 |
| 模型配置 | `/model` | AI 模型配置 |

## 五、交互规范

### 5.1 生成过程状态

| 状态 | 显示 |
|------|------|
| 等待中 | "正在准备生成..." + 骨架屏 |
| 生成中 | "AI 思考中..." + 动画（脉冲效果） |
| 成功 | 显示预览 + 绿色提示 + 质量评分 |
| 失败 | 红色错误提示 + 重新生成按钮 |

### 5.2 反馈机制

- 所有操作即时反馈（成功/失败 Toast）
- 长时间操作显示加载状态（Loading）
- 关键操作二次确认（如删除项目、删除页面）

### 5.3 快捷键

|--------|------|
